# Strategy Drop #1: The Content Engine

## Overview
The Content Engine is a high-volume content creation workflow designed to help solopreneurs publish consistently across multiple platforms without burning out. This strategy relies on **Content Pillars**, **Batching**, and **Systematic Repurposing**.

## The Workflow

### 1. Ideation & Pillars
*   **Define 3-5 Content Pillars:** These are the broad topics you want to be known for (e.g., Notion Tips, Automation, Freelance Life).
*   **The Idea Bank:** Never start from a blank page. Use a "Quick Capture" tool to dump every idea into your Notion Idea Bank immediately.

### 2. The Batching Method
*   **The Production Day:** Set aside one day a week (or a half-day) dedicated solely to content production.
*   **Phase 1: Outlining (30 mins):** Outline the main points for 4-5 core pieces of content.
*   **Phase 2: Drafting (2-3 hours):** Write the full drafts for those core pieces.
*   **Phase 3: Repurposing (1 hour):** Break down core pieces into smaller social media posts.

### 3. Systematic Repurposing
One long-form "Anchor" piece (Blog post or Newsletter) can be turned into:
*   2 Twitter Threads.
*   3 LinkedIn Posts.
*   5-7 Short Tweets/Insights.
*   1 Script for a Short-form Video (Reels/TikTok).

## Notion Template Specs: The Content Engine Extension

### 1. Anchor Content Database
*   **Properties:** Title, Status (Idea, Drafting, Repurposing, Published), Pillar, Publish Date, Link to Live Anchor.
*   **Template:** Anchor Post Template with sections for:
    *   Main Narrative
    *   Key Takeaways
    *   Repurposing Checklist

### 2. Social Snippets Database (Linked to Anchor)
*   **Properties:** Snippet Content, Platform (Twitter, LI, IG), Scheduled Date, Status, Anchor Piece (Relation).
*   **Views:** 
    *   **The Pipeline:** Board view by platform.
    *   **Scheduling Calendar:** Monthly view of all snippets.

### 3. Pillar Manager
*   **Database:** Content Pillars.
*   **Rollup:** See how many pieces of content have been published under each pillar to ensure balance.
