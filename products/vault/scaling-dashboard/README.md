# Scaling Dashboard - Notion Template Specs

This directory contains the technical specifications for **Strategy Drop #5: The Scaling Dashboard**.

## Purpose
To move solopreneurs from reactive task management to proactive strategic scaling. This template tracks both tactical monthly metrics (KPIs) and high-level quarterly objectives (OKRs) in one visual command center.

## Components
1. **[Monthly Metrics Tracker](./specs/monthly-metrics-tracker.md)**: Schema for tracking revenue, expenses, hourly rates, and conversion metrics.
2. **[Quarterly Goals (OKRs)](./specs/quarterly-goals-okrs.md)**: A strategic planning database with built-in progress visualization.
3. **[Scaling Dashboard Layout](./specs/dashboard-layout.md)**: The "Command Center" UI that aggregates data from the sub-databases.

## Key Feature: The Real Hourly Rate
Unlike standard accounting, this template calculates your "Real Hourly Rate" (Profit / Hours Worked). This is the single most important metric for a solopreneur to decide when to scale, raise prices, or hire a contractor.
