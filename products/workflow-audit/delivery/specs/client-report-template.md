# Client Audit Report: Template Specification

## Header Section
- **Client Name**
- **Effective Hourly Rate** (Used for ROI calculations)
- **Primary Goal** (e.g., "Scale to $20k/mo while working <20 hrs/week")

## Section 1: Executive Summary
- **Current Weekly Manual Work Load**: (Sum of Task Audit)
- **Potential Weekly Time Savings**: (Hours from 'Automate' + 'Remove')
- **Monthly Revenue Opportunity**: (Potential Savings * Effective Hourly Rate)

## Section 2: The ARC Audit Findings
A database view showing the top 5-10 tasks for each category:
- **Top 3 Automation Opportunities** (Quickest ROI)
- **Top 3 Processes to Remove** (Highest friction, lowest impact)
- **Top 3 Tasks to Centralize** (Reducing context switching)

## Section 3: Tech Stack Health Check
- **Current Subscriptions**: (List from Tech Stack Database)
- **Tool Recommendation**: (Add/Swap/Cancel recommendations)
- **Estimated SaaS Cost Savings**: (Monthly $)

## Section 4: Implementation Roadmap
- **Phased Projects**: (Rollup of Roadmap Database)
- **Phase 1 Deliverables**: (Explicit list of P0 tasks)

## Footer
- **Next Steps Checklist**
- **Implementation Tracker Link**
