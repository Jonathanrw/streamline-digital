# Affiliate Success Hub: Swipe File Library Spec

## 1. Swipe File Database [DB]
*Focus: Providing ready-to-use copy for partners.*

| Property Name | Type | Description |
| :--- | :--- | :--- |
| **Asset Name** | Title | e.g., "The Vault - Cold Email 1" |
| **Medium** | Select | `Email`, `LinkedIn`, `X (Twitter)`, `Instagram Story`, `Newsletter` |
| **Product** | Relation | Links to `Service Catalog` or Product Database |
| **Angle** | Select | `Save Time`, `Revenue Growth`, `Systematization`, `FOMO` |
| **Copy (Markdown)** | Text | The actual swipeable content |
| **Subject Line** | Text | (Optional) Recommended email subject lines |
| **Click-through Goal** | Text | Where they should send the traffic |

## 2. Views
- **By Product**: Grouped by which product the copy promotes.
- **Copy Gallery**: Card view for easy browsing of social snippets.
## 3. Seed Data (Pre-Populated Assets)
*Include these as default entries in the Hub for partners:*

### Email: The "Content Treadmill" (Problem/Solution)
- **Angle**: FOMO / Reclaiming Time
- **Copy**: "As a solopreneur, I know the 'content treadmill' feeling... [Includes breakdown of 3-stage batching system] ...reclaim 10+ hours of your week."

### LinkedIn: The "Admin Silent Killer"
- **Angle**: Revenue Growth
- **Copy**: "Administrative work is the silent killer of one-person businesses. If you're spending more than 2 hours a week updating CRM or tracking expenses, you're losing thousands in billable time..."

### Twitter: The "4-Hour Month" Thread
- **Angle**: High Result
- **Copy**: "I just batched 30 days of high-quality content in exactly 4 hours. No, I didn't use a massive agency. I used a system. 🧵"
