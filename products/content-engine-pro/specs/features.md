# Content Engine Pro: Key Features

## 1. The 3-Stage Batching System
*   **Stage 1: Idea Refinement.** Drag and drop ideas from the `Idea Bank` into the `Pipeline`.
*   **Stage 2: Core Writing.** Use the "Anchor Content" block to write the primary message once.
*   **Stage 3: Platform Multiplier.** Use dedicated views to tweak the anchor content for LinkedIn, X, and Instagram based on the Platform Master rules.

## 2. 365-Day Strategic Calendar
*   **Pre-filled Themes:** Each day is tagged with a "Theme of the Day" (e.g., *Monday Motivation*, *Tuesday Tutorial*) to eliminate blank-page syndrome.
*   **Slot-Based Scheduling:** Users don't just pick a date; they pick an available "slot" in the 365-day calendar.

## 3. Platform-Specific formatting
*   **X (Twitter) Thread Planner:** A dedicated space for thread-based writing with character count indicators.
*   **LinkedIn Hook Analyzer:** A visual checklist of the top 5 hook types to ensure high click-through rates.
*   **Instagram Caption Optimizer:** Auto-formatting for clean line breaks and a "Hashtag Cloud" manager.

## 4. Gateway to The Vault
*   **Vault Integration:** A dedicated dashboard section showing "Locked" features (e.g., Automated Social Echo, Advanced Analytics) that are available in the full Vault membership.
*   **Upgrade Path:** Strategically placed call-to-actions (CTAs) within the template to drive conversion to the high-ticket products.

## 5. Pre-Built Frameworks & Pillars
*   **The Framework Library:** A curated set of high-converting writing frameworks (AIDA, PAS, The Bridge) integrated directly into the drafting page.
*   **Strategic Pillar Tracking:** Users can see exactly which topics they are over-posting or under-posting, ensuring a balanced brand authority.
